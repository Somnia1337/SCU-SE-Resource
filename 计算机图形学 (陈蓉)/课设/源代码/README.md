# [three-d-scene](https://github.com/Somnia1337/three-d-scene)

100% 使用 Rust 以及 `three-d` crate 实现的、可用键鼠交互的、提供 GUI 参数控制面板的、高帧率的 3D 场景建模。

> [!IMPORTANT]
> 这只是一个原型系统。
